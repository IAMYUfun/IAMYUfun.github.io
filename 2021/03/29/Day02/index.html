Java入门及环境搭建

Java优势及特性

  1. 简单性
  2. 面向对象
  3. 可移植性
  4. 高性能
  5. 分布式
  6. 动态性
  7. 多线程
  8. 安全性
  9. 健壮性

Java三大版本

JavaSE ：标准版

JavaEE：E企业级开发

JavaME： 嵌入式开发

JDK、JRE、JVM

JDK : Java Development Kit

JRE : Java Runtime Environment

JVM : JAVA Virtual Machine

卸载JDK

  1. 删除Java的安装目录
  2. 删除JAVA_HOME
  3. 删除path下关于Java的目录
  4. Java -version

安装JDK

  1. 浏览器搜索JDK8，找到下载地址

  2. 同意协议

  3. 下载电脑对应版本

  4. 双击安装JDK

  5. 记住安装路径

  6. 配置环境变量

    1. 我的电脑–>右键–>属性
    2. 环境变量–>JAVA_HOME，变量值为 jdk 的安装路径。
    3. 新建变量名：CLASSPATH ，变量值： .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar
    4. 配置path变量，%JAVA_HOME%\bin
